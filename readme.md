# Tyky

> O **oceano** começa com uma **gota**.

_Inspirado por [Anki][anki]._


Tyky é uma plataforma de aprendizado baseada em repetição espaçada, desenvolvida para ajudar usuários a transformar estudo em memória de longo prazo. Através de revisões inteligentes e progressivas, o sistema apresenta conteúdos no momento ideal para reforçar o conhecimento, reduzindo o esquecimento e aumentando a retenção.

## Objetivo

Tornar o aprendizado mais eficiente, acessível e duradouro, permitindo que qualquer pessoa construa conhecimento de forma consistente — uma **gota** por vez.

## Principais características

- 📚 Criação e gerenciamento de flashcards
- 🧠 Repetição espaçada baseada em desempenho
- 🔄 Revisões programadas automaticamente
- 🎯 Foco em retenção de longo prazo

## Algoritmo de Repetição Espaçada

O Tyky utiliza o algoritmo SM-2 (SuperMemo-2) como base para o agendamento das revisões, um dos algoritmos de repetição espaçada mais conhecidos e amplamente utilizados para otimizar a retenção de conhecimento ao longo do tempo.

O objetivo do algoritmo é apresentar cada card no momento ideal para revisão, equilibrando retenção e eficiência no aprendizado.
